# Aamir Portfolio Website

Personal portfolio site for **Aamir Saleem** with modern UI, animations, and responsive layout.

## Live URL
- https://aamir0011.github.io/aamir-portfolio-website/

## Repository
- https://github.com/aamir0011/aamir-portfolio-website

## Features
- Professional profile sections (summary, experience, education, skills, contact)
- LinkedIn + GitHub quick links
- Smooth scrolling and theme toggle
- Animated counters and reveal effects
- Fancy motion pack (particles, progress bar, tilt cards, back-to-top)

## Run locally
```bash
git clone https://github.com/aamir0011/aamir-portfolio-website.git
cd aamir-portfolio-website
python3 -m http.server 8080
```
Open: http://localhost:8080
